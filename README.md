# EVE Multitool

A React + TypeScript + MUI app scaffold for EVE Online courier contract discovery.

## Features

- Filter courier contracts by collateral, cargo volume, route type, and location.
- Sort results by table headers.
- Select an attractivity scoring method with descriptions.

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

## Notes

This starter app uses sample contract data. The contract table and filter UI are ready for integration with EVE Online public APIs.
